# 资源文件说明

## 图标文件

打包应用需要以下图标文件：

### Windows 图标
- `icon.ico` - 256x256 或更大尺寸的 Windows 图标

### macOS 图标
- `icon.icns` - macOS 图标文件（包含多种尺寸）

### 通用图标
- `icon.png` - 512x512 或更大尺寸的 PNG 图标

## 生成图标

可以使用以下在线工具生成：
1. [ICO Convert](https://icoconvert.com/) - PNG 转 ICO
2. [iConvert Icons](https://iconverticons.com/online/) - 生成 ICNS
3. [Electron Icon Generator](https://www.electron.build/icons.html)
